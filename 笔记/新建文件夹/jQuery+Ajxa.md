# jQuery+Ajax
## 如何使用jQuery
>1. 下载地址: http://code.jquery
>
>2. 引用下载的jQuery库
>
>   ```html
>   <head>
>       <meta charset="UTF-8">
>       <title>01-初识jQuery</title>
>       <script src="./jquery-3.4.1.min.js"></script>
>   </head>
>   ```
>
>   3. 编写jQuery代码
>
>      ```javascript
>      <script>
>          //1. 原生JS的固定写法
>          window.onload = function() {
>  
>      };
>      
>          //2.jQuery的写法
>          $(document).ready(function () {
>              alert("hello world");
>          });
>      </script>
>      ```
